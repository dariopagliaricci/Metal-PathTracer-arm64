# Changelog

All notable changes to this project will be documented here.

## [v1.0.0] — 2025-11-19
**Initial public release targeting Apple Silicon (arm64).**

### Added
- Hardware Ray Tracing (Metal TLAS/BLAS) backend.
- Software BVH path tracer (reference implementation).
- Progressive renderer with temporal accumulation.
- HDR environment lighting:
  - Importance sampling (alias table)
  - MIS for diffuse/specular domains
  - Environment rotation / intensity overrides
- Physically based material model:
  - Lambertian diffuse
  - Conductor metal (η/k)
  - Dielectric (glass) with absorption & coat
  - Plastic (clear coat, tint, absorption)
  - Car paint (base + flakes)
  - Subsurface scattering (random-walk BSSRDF)
  - Diffuse emissive lights
- OBJ / PLY mesh support.
- Intel® Open Image Denoise (OIDN 2.3.3) integration.
- Firefly management: specular tail clamp + global clamp.
- Tonemappers: Linear, ACES, Reinhard, Hable.
- Internal render scale (0.5×–2.0×).
- GUI (`PathTracer.app`) using ImGui + ImGuizmo.
- EXR export:
  - Linear HDR EXR writer
  - Optional multilayer EXR (RGB + sampleCount)
  - Automatic timestamped filenames under `./renders/`
- Headless renderer (`PathTracerHeadless`) with:
  - `--scene`, `--width/height`, `--sppTotal`, `--enableSoftwareRayTracing`, `--envIntensity`, `--envRotation`, `--format`, `--output`
- Minimal asset pack embedded in repo (Cornell + RTOW scenes + two HDRIs).

### Fixed
- Unified shading normals between Software BVH and Hardware RT.
- Corrected biasing & self-hit issues under HWRT.
- Stability fixes in ImGui rendering at HiDPI scales.
- Deterministic RNG seeding for headless rendering.

### Known Limitations
- Minor glass-parity differences between HWRT and SWRT.
- EXR export uses autogenerated filenames (no Save-As dialog yet).
- Project targets **Apple Silicon only** (OIDN + Metal RT).
- Golden test scripts remain internal and are not included in the public repo.
